This project is a drive-style file manager web application built to simulate the core functionality of a cloud storage platform. First, users can create folders and upload files through a simple and interactive interface. Each action—adding or deleting files and folders—updates the UI instantly using JavaScript DOM manipulation. The application then stores all file and folder data in Local Storage so that the information persists even after the page is refreshed. The layout is designed to be clean and responsive, making navigation between folders smooth and intuitive. Through this project, I focused on understanding how real-world file management systems work, how frontend and storage logic interact, and how to build a structured, user-friendly web application using HTML, CSS, and JavaScript.
